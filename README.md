# 跟爺爺學中文 · Chinesisch lernen mit Opa

給德國小孫女的中文互動遊戲。單一網頁，平板／手機／電腦都能玩，可離線。

## 怎麼打開玩
直接用瀏覽器打開 `index.html`（雙擊即可）。
要傳到德國：把整個「中文遊戲」資料夾壓縮寄過去（已附 `跟爺爺學中文.zip` 在桌面），對方解壓後打開 `index.html`。

## 在手機／平板上像 App 一樣（PWA）
本遊戲已打包成可安裝的網頁 App：有 App 圖示、全螢幕、離線可玩。
**重點：要有「網址」才能變成真正的 App（離線快取需要 https）。** 兩種方式：

1. **放到免費網址（推薦）** — 例如 GitHub Pages / Netlify。對方用手機開網址 →
   - iPhone/iPad（Safari）：分享鈕 → **加入主畫面**
   - Android（Chrome）：選單 → **安裝應用程式 / 加到主畫面**
   - 之後從主畫面圖示打開 = 全螢幕、無網址列、離線可玩，跟 App 一樣
2. **只傳檔案** — 對方解壓用瀏覽器打開也能玩（橫豎自適應），但不是桌面圖示 App、離線快取不啟用。

> 圖示在 `icons/`，設定在 `manifest.json`，離線快取在 `sw.js`。

## 目前內容
- 主題（首頁可切換）：🐶 動物 10、🎨 顏色 10、🔢 數字 1-10、👨‍👩‍👧‍👦 家人 8、🍎 食物 10
- **L1 認識**：點圖卡 → 聽中文發音，看 中文／拼音／德文；點藍色德文字唸德文
- **L2 聽聲音找動物**：聽中文 → 在 3 張圖裡點對的；題目欄有中文＋德文兩顆發音鈕
- **L3 找一找**：6 隻裡找出指定動物（難一階）
- **🏆 收集冊**：答對過的動物會被收集，點右上角 ⭐ 打開看
- 發音：中文目前是**電腦暫代發音**（錄音放進去後自動改用爺爺的聲音）；德文是電腦德語語音
- 🎵 **音樂**：首頁三顆音樂鈕，是小孫女喜歡的歌，點一下播放、再點一次停止（檔案在 `music/`，可自行替換 `song1~3.m4a`）

## Phase 2：換成爺爺的聲音

**最簡單的方法 → 打開 `recorder.html`（錄音小幫手）**
1. 開啟麥克風（按允許）→ 每列按「● 錄音」念詞 →「停止」
2. 「▶ 試聽」確認 →「⬇ 下載」
3. 全部錄完，把「下載」資料夾裡的檔案**移到本資料夾的 `audio/`**
4. 重開遊戲，就會用爺爺的聲音了 🎉

> 錄音小幫手存出的檔名已經是正確的（如 `dog.m4a` / `dog.webm`，看你的瀏覽器）。
> 遊戲會自動找 `audio/` 裡同名的 mp3 / m4a / webm / ogg / wav，找不到才用電腦發音。

> 錄音小幫手現在涵蓋全部 5 個主題（動物／顏色／數字／家人／食物，共 48 個詞），下載的檔名都已對好。
